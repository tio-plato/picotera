# 脚本 ID 可指定与修改

为脚本 ID 增加可指定与可修改的能力：

- 创建脚本时，可以指定脚本 ID，作为 slug 使用；不指定则默认随机生成一个。
- 编辑脚本时，可以修改脚本 ID（即重命名 slug）。

## 补充约定（澄清后确定）

- **Slug 格式**：脚本 ID 必须匹配 `^[a-zA-Z0-9_-]+$`（大小写字母、数字、下划线、短横线），非空。严格校验，不做宽松归一化。
- **改 ID 实现方式**：编辑时通过原地 `UPDATE script SET id = ...` 修改主键，请求 URL 仍走旧 ID 的 `PUT /scripts/{id}`，请求体携带新 ID。
- 脚本 ID 当前在系统内没有任何外部引用（请求、日志、其他表均不引用），修改 ID 是安全的。
