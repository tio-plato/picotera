# Proposal

探索如何引入 Nuxt UI 作为 dashboard 的组件库，同时保留当前的设计语言。使用当前 dashboard 的设计语言和变量覆盖 Nuxt UI 的主题。

补充澄清：不希望把现在的 dashboard 组件改用 Nuxt UI 里的组件。当前大部分组件仍然自己写、自己维护。Nuxt UI 只作为未来按需使用的补充组件来源；如果以后确实需要其中的一小部分组件，再通过 PicoTera 自己的 wrapper 引入，并继续遵循当前设计语言和变量。

参考资料：https://ui.nuxt.com/raw/docs/getting-started/theme/design-system.md
